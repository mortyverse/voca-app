# Noun Gravity - Countable vs Uncountable Nouns

A simple yet beautiful application to help you master English noun countability.

## Features

1.  **Vocabulary Management**: Add English words and mark them as Countable (C) or Uncountable (U).
2.  **Practice Mode**: Test yourself one word at a time with instant feedback.
3.  **Full Test Mode**: Take a comprehensive test on all your saved words and get a score.
4.  **Auto-Save**: Your vocabulary list is automatically saved to your browser's Local Storage.

## How to Run

1.  Ensure you have Node.js installed.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the application.
4.  Open `http://localhost:5173` in your browser.

## Technologies

- React
- Vite
- Modern CSS (Dark/Light mode support)
